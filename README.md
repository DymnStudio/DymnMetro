# Metro Launcher

A Windows Phone / Metro inspired Android launcher starter built with Kotlin and Jetpack Compose.

## What is included

- HOME intent registration so Android can offer this app as a launcher.
- Installed-app discovery through `PackageManager`.
- Tile based home screen inspired by Windows Phone.
- Metro style app menu with search and alphabetical sections.
- Direct app launching by component name.

## Open and run

1. Open this folder in Android Studio.
2. Let Gradle sync the project.
3. Run the `app` configuration on an Android device or emulator.
4. Press the device Home button and choose Metro Launcher.
